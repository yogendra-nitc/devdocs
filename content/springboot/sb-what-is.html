Spring Boot is an opinionated framework built on top of Spring that simplifies Java application configuration and deployment. It reduces boilerplate setup and helps you build production-ready applications quickly.

Why Spring Boot?

  • Auto-configuration – Spring Boot automatically configures many common Spring components.
  • Starter dependencies – Use preconfigured dependency bundles for web, data, security, and more.
  • Embedded server – Run your app as a standalone JAR with Tomcat, Jetty, or Undertow built in.

Example

Java 
@SpringBootApplication
public class Application {

    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}
Spring Boot vs Spring Framework
Spring Boot is built on the Spring Framework and adds auto-configuration, starter POMs, and embedded servers for quicker app setup.
