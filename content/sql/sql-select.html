The SELECT statement is the first command you use when learning SQL. It retrieves rows from one or more tables in a database.

SELECT Syntax

SQL 
SELECT column1, column2
FROM employees
WHERE department = 'Engineering'
ORDER BY last_name ASC;

Example

This query returns employee names and salaries for the Engineering department, sorted by last name.

Note: Use * to select all columns, but prefer specifying columns for better performance and clarity.
