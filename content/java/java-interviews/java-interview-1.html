The classic first program — printing "Hello, World!" — teaches you the minimal structure of a Java program.

1. What is a class in Java?

A class is a blueprint or template used to create objects. It defines the properties (fields) and behaviors (methods) of objects.

Example:

Java 

    class Student {
    String name;
    int age;

    void display() {
        System.out.println(name + " " + age);
    }
  }

2. What is an object?

An object is an instance of a class. It occupies memory and can access the class's fields and methods.

Example:

Java 

    Student s = new Student();
    s.name = "Rahul";
    s.age = 20;
    s.display();

3. What is the difference between a class and an object?

- A class is a blueprint that defines properties and methods and no memory is allocated until objects are created.
- An object is an instance of a class with memory allocated for its data and it uses the properties and methods defined by the class.

Example:

Java 

    // Class definition
    class Student {
        String name;
        int age;

        void display() {
            System.out.println(name + " " + age);
        }
    }

    // Object creation
    Student s = new Student();
    s.name = "Rahul";
    s.age = 20;
    s.display();

4. What is a constructor?

A constructor is a special method that initializes an object. It has the same name as the class and has no return type.

Example:

Java 

    class Student {
        Student() {
            System.out.println("Constructor called");
        }
    }

5. What are the types of constructors?

* Default Constructor
* No-Argument Constructor
* Parameterized Constructor
* Copy Constructor (user-defined pattern in Java)

Example:

Java 

    class Student {
        String name;
        Student(String n) {
            name = n;
        }
    }

6. What is the `this` keyword?

`this` refers to the current object. It is used to distinguish instance variables from local variables and to call another constructor.

Example:

Java 

    class Student {
        String name;
        Student(String name) {
            this.name = name;
        }
    }

7. What is the difference between instance variables and local variables?

Instance variables are declared inside a class but outside any method, constructor, or block. They belong to an instance of the class and have default values.
Local variables are declared inside a method, constructor, or block and exist only during the execution of that method. They must be initialized before use.

8. What are static members in a class?

Static members belong to the class rather than individual objects. They can be accessed without creating an object. Static variables are shared among all instances of the class, and static methods can only access static variables and other static methods.

Example:

Java 

    class Student {
        static String college = "ABC College";
        static void showCollege() {
            System.out.println(college);
        }
    }

    Student.showCollege();

9. What is method overloading?

Method overloading means defining multiple methods with the same name but different parameter lists.

Example:

Java 

    class Calculator {
        int add(int a, int b) {
            return a + b;
        }
        int add(int a, int b, int c) {
            return a + b + c;
        }
    }

10. What is encapsulation?

Encapsulation is the practice of hiding data by making fields private and providing public getter and setter methods to access them.

Example:

Java 

    class Employee {
        private int salary;
        public void setSalary(int salary) {
            this.salary = salary;
        }
        public int getSalary() {
            return salary;
        }
    }
