The classic first program — printing "Hello, World!" — teaches you the minimal structure of a Java program.

The Code

Java 
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Running the Program

Terminal 
# Step 1: Compile
javac HelloWorld.java

# Step 2: Run
java HelloWorld

# Output
Hello, World!
