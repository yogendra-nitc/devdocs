Bubble Sort is one of the simplest sorting algorithms. It repeatedly compares adjacent elements and swaps them when they are in the wrong order.

How It Works

Each pass moves the largest unsorted element to the end of the list, like bubbles rising to the surface.

Implementation

Java 
public static void bubbleSort(int[] arr) {
    int n = arr.length;
    for (int i = 0; i < n - 1; i++) {
      for (int j = 0; j < n - i - 1; j++) {
        if (arr[j] > arr[j + 1]) {
          int temp = arr[j];
          arr[j] = arr[j + 1];
          arr[j + 1] = temp;
        }
      }
    }
}
Note: Bubble Sort is easy to understand, but it is not efficient for large datasets. Use it for learning or small arrays.
